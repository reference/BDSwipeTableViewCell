# BDSwipeTableViewCell

Subclass for MGSwipeTableViewCell

## Requirements

* Requires iOS 9.0 or later

## Installation

BDSwipeTableViewCell is available through [CocoaPods](https://cocoapods.org/pods/BDSwipeTableViewCell). To install it, simply add the following line to your Podfile:

```
use_frameworks!

pod 'BDSwipeTableViewCell'

```

## Usage

```
#import <BDSwipeTableViewCell/BDSwipeTableViewCell.h>

```
